# Dinkbit Payme

Important: Alpha version, do not use in production.

Based on [Active Merchant](http://github.com/Shopify/active_merchant) for Ruby and [Aktive-Merchant](https://github.com/akDeveloper/Aktive-Merchant) for PHP

Supported Gateways:
* Conekta
* Conekta Oxxo
* Conekta Bank
* Banwire Recurrent
* Paypal Express (soon)
